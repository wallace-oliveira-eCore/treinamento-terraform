## Dia 01
### Introdução ao Git

### Comandos básicos
- git init -> inicializa repositório local
- git status -> mostra status, se tem arquivo novo, para comitar
- git add -> adiciona arquivo a área de staging, para ser comitado
- git commit -> grava a modificação no histórico do git
- git log -> mostra o histórico de commits
- git branch -> mostra os branches
- git branch X -> criar branch X
- git checkout X -> muda para branch X
- git branch -D X -> apata branch X
- git reset HEAD~X -> reverte x commits e mantem as modificações
- git reset --hard HEAD~X -> reverte x commits e descarta as modificações
- git merge X -> faz o merge do branch X no branch atual
- git config merge.ff false -> muda a estratégia de merge para criar um merge commit
- git rebase X -> faz o rebase do branch X no branch atual; insere os commits do outro branch no seu branch e coloca SEMPRE os seus commits no HEAD sem criar merge commit
- git pull -> traz as modificações do repositório remoto
